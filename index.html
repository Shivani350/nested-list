      1. HTML (hypertext markup language)
          2. Versions of HTML
        1.     HTML 1.0 :-       The first version of HTML, used from 1989 to 1994, with only 20 elements.
        2.     HTML 2.0 Released in 1995, this version introduced forms and tables, which expanded the possibilities for web content.
        3.     HTML 3.2 :-      Released in 1997, this version added support for tables, applets, and text flow around images.
        4.     HTML 4.01 :-      Released in April 1998, this version was a major overhaul of the HTML standards.
        5.     XHTML 1.0 :-      A reformulation of HTML as an XML application, XHTML 1.0 is technically stricter than HTML 4.01.
        6.     HTML 5 :-       A recent and updated version of HTML, HTML 5 has changed or removed many HTML tags for better functionality.
      2. CSS (cascading style sheets)
